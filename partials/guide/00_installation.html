Quick Start

To get started the following prerequisites are to be installed.

1) Install NodeJs

Node.js is a runtime environment and a library for running applications written in JavaScript outside the browser (for example, on the server).

Download

2) Install Git

Download

3) Install Grunt and Bower

$ npm install -g grunt-cli bower

4) Clone the pro-angular-yeoman repository to your local machine

$ git clone https://github.com/Ashwin2488/pro-angular-yeoman.git

This will create your application skeleton with some existing controllers and services.
