Angular Base Skeleton

AngularJs skeleton with recommended folder structure.

Features

  • Scaffolds google recommended app structure for scalable AngularJs application

  • A common config file to manage your service URLS ,app version,release mode,app specific global configurations 

    {
  "app": {
    "version": "0.0.1",
    "mode": "staging",
    "name": "proAngular",
    "lang": "es",
    "staging": {
      "serviceUrl": "",
      "pollInterval": "",
      "serviceTimeout": ""
    },
    "production": {
      "serviceUrl": "",
      "pollInterval": "",
      "serviceTimeout": ""
    }
  },
  "services": {
    "matchList": "",
  }
}
  • Jasmine unit test integrated with karma runner.
  • code coverage for unit tests to identify untested and unused code in your application
